# 广州理工学院校园网自动登录工具
本项目最初受到校园网登录脚本的启发，经过完全重写和功能扩展，现已成为一个功能完善的自动化登录解决方案。
🐒 猴子都能看懂的校园网自动登录脚本使用说明书
本文档专为猴子设计，保证0基础也能学会！
---
📦 一、这是什么东西？
这是一个校园网自动登录小工具，就像你家里的自动门锁：
· 以前：每天回家要用钥匙开门（手动输入账号密码登录校园网）
· 现在：走到门口，门自动打开（双击一下，自动帮你登录）
一句话总结：一个小程序，帮你自动登录学校网络，不用每次输账号密码！

📁 二、文件清单（就像工具箱里的工具）
二、文件清单
将这些文件类比为厨房厨具：
文件作用类比厨具
CampusNet.bat	主开关，双击启动	微波炉，一键热饭
FirstTimeSetup.bat	首次设置运行	菜谱，指导做菜
CampusNet_Login.ps1	核心程序	大厨，实际烹饪
CampusNet_Silent.ps1	开机自动登录	小工，默默帮忙
CreateDesktopShortcut.ps1	创建桌面快捷方式	电话快捷拨号
CampusNet_Config.json	存储账号密码（自动生成）	密码锁，保存秘密
CampusNet_Log.txt	记录登录情况（自动生成）	日记本，记录日常
💡 小贴士：您只需关注前两个 .bat 文件，其余交给电脑。

🎬  三、使用步骤（从零开始，一步一步）
第1步：准备材料（就像准备做菜）
1. 找个地方放文件：
   · 在桌面新建一个文件夹，叫校园网登录
   · 把所有文件都放进去（就像把调料放调料盒）
2. 检查文件：
   · 确保这些文件都在同一个文件夹里
   · 就像炒菜时，油盐酱醋都在灶台上，不能东一个西一个

第2步：第一次使用（就像第一次用微波炉）
1. 双击 FirstTimeSetup.bat
   · 会弹出一个黑色窗口
   · 这是正常的，别害怕！
2. 按提示操作：
   ```
   第一步：输入学号
   请输入您的学号：12345678912345
   
   第二步：输入密码
   请输入您的校园网密码：123456
   ```
   · 注意：输密码时不会显示***，这是为了保护你的隐私
   · 正常输就行，输完按回车
3. 等待检测：
   · 程序会自动检查你的网络
   · 显示类似这样就是成功了：
     ```
     找到网络适配器：以太网 2
     IP地址：0.0.0.0
     MAC地址：00-00-00-00-00-00
     连接类型：000
     ```

第3步：日常使用（就像每天热饭）
1. 连接网络：
   · 插上网线（如果用有线）
   · 或连接WiFi（如果用无线）
2. 双击 CampusNet.bat
   · 等几秒钟
   · 看到"登录成功！"就可以上网了
3. 验证：
   · 打开浏览器
   · 访问 www.baidu.com或其余网址
   · 能打开网页就是成功了！

🎯 四、不同场景使用方法

场景一：电脑小白（最简单的方法）
1. 把文件夹放在桌面
2. 把 CampusNet.bat 发送到桌面快捷方式
3. 每次上网前，双击桌面的快捷方式

场景二：想更省事（开机自动登录）
1. 创建快捷方式：
   · 双击 CreateDesktopShortcut.ps1
   · 桌面上会出现一个"校园网登录"的快捷方式
2. 设置开机启动：
   · 按 Win + R（Win就是键盘上的Windows图标键）
   · 输入 shell:startup 按回车
   · 把 CampusNet_Silent.ps1 拖到这个文件夹里
   💡 小猴理解：就像设置闹钟，每天早上自动响铃。

场景三：多个同学用同一台电脑
如果需要多个账号切换：
1. 运行 FirstTimeSetup.bat 重新设置
2. 输入新的学号和密码
3. 旧配置会被新配置覆盖

🔧 五、常见问题和解决办法（就像家电故障处理）
问题1：双击后闪退，什么都没发生
可能原因：文件编码问题
解决办法：
1. 右键点击 CampusNet.bat → "编辑"
2. 看看里面有没有乱码（奇怪的符号）
3. 如果有，用我给的正确版本重新创建

问题2：显示"找不到文件"
可能原因：文件没放在一起
解决办法：归档啦
```
错误示例：
C:\Users\用户\Desktop\校园网登录\CampusNet.bat
C:\Users\用户\Downloads\CampusNet_Login.ps1  ← 错！在不同文件夹

正确示例：
C:\Users\用户\Desktop\校园网登录\CampusNet.bat
C:\Users\用户\Desktop\校园网登录\CampusNet_Login.ps1 ← 对！在同一文件夹
```

问题3：登录失败，显示"ret_code:8"
可能原因：
1. 账号已在其他地方登录
2. IP地址不对
解决办法：
1. 先退出其他地方：
   · 用手机或其他电脑登录校园网页面
   · 找到"注销"或"下线"按钮，点击
2. 重新获取IP：
   · 按 Win + R 输入 cmd 按回车
   · 输入这两条命令，每条输完按回车：
     ```
     ipconfig /release
     ipconfig /renew
     ```

问题4：能登录但不能上网
可能原因：需要等一会儿
解决办法：
1. 等待1-2分钟
2. 重启浏览器
3. 如果还不行，重新运行一次登录脚本

问题5：显示乱码（奇怪的文字）
可能原因：编码问题
解决办法：
在CampusNet_Login.ps1 文件开头添加：
```powershell
[Console]::OutputEncoding = [System.Text.Encoding]::UTF8
```

📝 六、给猴子的特别提醒
安全注意事项：
1. 密码安全：脚本会把密码加密保存，但还是要保管好文件夹
2. 个人隐私：不要在公用电脑保存你的配置
3. 定期检查：如果学校改了登录方式，需要更新脚本，请在issue中告知

使用技巧：
1. 先连接，后登录：一定先插网线/连WiFi，再运行脚本
2. 看颜色判断：
   · ✅ 绿色字 = 成功
   · ⚠️ 黄色字 = 警告（可能有点小问题）
   · ❌ 红色字 = 错误（需要处理）
3. 日志有用：如果出问题，打开 CampusNet_Log.txt 看看发生了什么
文件备份：
重要的两个文件：
1. CampusNet_Config.json ← 你的账号密码在这里
2. CampusNet_Login.ps1 ← 主程序
建议定期备份到U盘。

📚 十、技术原理（给好奇的猴子）
脚本做了什么：
1. 找身份证：获取电脑的MAC地址（像身份证号）
2. 找门牌号：获取电脑的IP地址（像家庭住址）
3. 打电话：把这些信息+账号密码发给学校服务器
4. 等回信：接收服务器的回复
5. 开门：如果正确，就让你上网

为什么安全：
1. 本地存储：密码存在你自己电脑上
2. 简单加密：密码被转换成了看不懂的代码
3. 只发必要信息：只发送登录需要的信息
