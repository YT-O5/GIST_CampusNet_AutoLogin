📖 简介
本项目最初受到校园网登录脚本的启发，经过完全重写和功能扩展，现已成为一个功能完善的自动化登录解决方案。

一句话总结：一个让校园网登录像微波炉热饭一样简单的小工具！

🎯 核心功能
✅ 一键登录：双击即可完成校园网认证

✅ 自动识别：智能检测网络适配器状态

✅ 安全存储：本地加密保存账号信息

✅ 详细日志：完整记录每次登录情况

✅ 多场景适配：支持有线/无线网络连接

✅ 开机自启：可设置为开机自动登录

🐒 特别说明：猴子都能看懂！
本文档专为零基础用户设计，保证小白也能轻松上手！
就像使用微波炉不需要懂原理，使用本工具也不需要懂技术。

📁 工具箱（文件说明）
文件	作用	通俗比喻
CampusNet.bat	主程序入口	微波炉开关
FirstTimeSetup.bat	首次配置向导	使用说明书
CampusNet_Login.ps1	核心登录逻辑	微波炉核心
CampusNet_Silent.ps1	静默登录版本	定时加热功能
CreateDesktopShortcut.ps1	创建桌面快捷方式	快捷拨号键
CampusNet_Config.json	存储账号密码（自动生成）	密码保险箱
CampusNet_Log.txt	登录记录（自动生成）	使用日记本
💡 小贴士：您只需关注前两个 .bat 文件，其他文件交给程序自动管理。

🚀 快速开始（3分钟上手）
第1步：准备材料
在桌面新建文件夹，命名为 校园网登录

将所有文件放入该文件夹（就像把调料放调料盒）

确保所有文件都在同一文件夹内

第2步：首次配置
双击 FirstTimeSetup.bat

按提示输入信息：

bash
请输入您的学号：你的学号
请输入校园网密码：你的密码（输入时不会显示***，正常输入即可）
程序会自动检测网络，看到类似信息即为成功：

text
✓ 找到网络适配器：以太网
✓ IP地址：10.x.x.x
✓ MAC地址：00-00-00-00-00-00
第3步：日常使用
连接校园网（插网线或连接WiFi）

双击 CampusNet.bat

看到"✅ 登录成功！"即可上网

打开浏览器测试：访问 www.baidu.com

🎮 使用场景指南
场景一：电脑小白（最简单）
将文件夹放在桌面

右键 CampusNet.bat → "发送到" → "桌面快捷方式"

需要上网时，双击桌面快捷方式

场景二：追求便捷（推荐）
运行 CreateDesktopShortcut.ps1 创建桌面快捷方式

设置开机自动登录：

按 Win + R 键

输入 shell:startup 回车

将 CampusNet_Silent.ps1 放入此文件夹

任务计划启动

场景三：多人共用电脑
不同用户请使用各自独立的文件夹

或运行 FirstTimeSetup.bat 重新配置（会覆盖旧配置）

🔧 故障排除手册
❌ 问题1：双击后闪退
可能原因：文件编码问题
解决方法：

确保所有文件在同一文件夹

右键 CampusNet.bat → "编辑"

检查是否有乱码，如有则重新下载

❌ 问题2：显示"找不到文件"
错误示例（× 错误）：

text
C:\Desktop\校园网登录\CampusNet.bat
C:\Downloads\CampusNet_Login.ps1   ← 文件分散在不同位置
正确示例（✓ 正确）：

text
C:\Desktop\校园网登录\CampusNet.bat
C:\Desktop\校园网登录\CampusNet_Login.ps1   ← 所有文件在一起
❌ 问题3：登录失败"ret_code:8"
原因：账号在其他设备登录
解决：

用手机/其他电脑登录校园网页面

找到"下线"或"注销"按钮

返回本机重新登录

❌ 问题4：能登录但无法上网
解决步骤：

等待1-2分钟（校园网有时延迟）

重启浏览器

重新运行登录脚本

❌ 问题5：显示乱码
解决方法：
在文件开头添加（技术人员可操作）：

powershell
[Console]::OutputEncoding = [System.Text.Encoding]::UTF8
⚠️ 重要安全提醒
密码安全

不要在公用电脑上保存个人配置

定期修改校园网密码

🛠️ 技术原理（好奇者可见）
脚本工作原理
身份识别：获取电脑MAC地址（类似身份证）

地址确认：获取当前IP地址（类似家庭住址）

发送请求：将认证信息发送至校园网服务器

接收响应：获取登录结果

网络连通：成功则建立网络连接

安全保障
本地存储：所有敏感信息仅存于本地

基础加密：密码进行编码保护

最小权限：仅发送必要认证信息

透明日志：所有操作均有记录
