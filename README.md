# 🏫 广州理工学院校园网自动登录工具

## ✨ 特性

- 🔒 **自动登录**：一键连接校园网
- 🎯 **智能检测**：自动识别有线/无线网络
- 📝 **配置向导**：友好的首次设置界面
- 📊 **日志系统**：记录所有登录活动
- 🔄 **断线重连**：网络中断后自动恢复
- 🖥️ **多用户支持**：适用于不同学生账号

## 📥 快速开始

1. 下载最新版本
2. 运行 `FirstTimeSetup.bat`
3. 输入学号和密码
4. 以后使用 `CampusNet.bat` 一键登录
5. 键盘输入 `win+R` 后输入 `taskschd.msc` 打开任务计划启动后新建计划

## 🛠️ 技术栈

- **语言**: PowerShell 5.1+
- **平台**: Windows 10/11
- **依赖**: 无外部依赖
