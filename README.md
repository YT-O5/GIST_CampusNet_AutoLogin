# 🏫 广州理工学院校园网自动登录工具

## ✨ 特性

- 🔒 **自动登录**：一键连接校园网
- 🎯 **智能检测**：自动识别有线/无线网络
- 📝 **配置向导**：友好的首次设置界面
- 📊 **日志系统**：记录所有登录活动
- 🔄 **断线重连**：网络中断后自动恢复
- 🖥️ **多用户支持**：适用于不同学生账号

## 📥 快速开始

1. 下载最新版本
2. 运行 `FirstTimeSetup.bat`
3. 输入学号和密码
4. 双击 `CampusNet.bat` 一键登录
5. 键盘输入 `win+R` 后输入 `taskschd.msc` 打开任务计划启动后新建计划

## 🛠️ 高级功能快速启用

- **开机自启动**:
- # 方法一：简单版（适合大多数人）
1. 按 Win + R 输入 taskschd.msc
2. 创建基本任务
3. 触发器选"计算机启动时"
4. 操作选"启动程序"
5. 程序填 CampusNet_Silent.ps1
- # 方法二：专业版（适合想折腾的）

- **平断网自动重连**: 
📡 已内置！无需额外设置
工作原理:
  1. 程序每30分钟检查一次网络
  2. 如果断网，自动尝试重连
  3. 最多重试3次
  4. 重连成功会有通知

手动触发: 再次运行 CampusNet.bat
- **多校区支持（存疑）**:
- 适用: 广州理工学院不同校区
- 配置:
  1. 打开 CampusNet_Config.json
  2. 修改服务器地址:
     - 主校区: http://10.0.10.252:801/eportal/
     - 分校区: （根据实际填写）（惠州校区的我没人脉以及路径尝试，但是理论上可以）
  3. 保存文件重新登录
注意: 不同校区可能需要不同参数
